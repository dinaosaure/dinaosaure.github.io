IMCTrack (2025)

Kotlin Android App (UPM Semester)

IMCTrack is an Android mobile application developed in Kotlin during my international semester at UPM. The app applies advanced mobile development concepts and focuses on user experience and data management.

  • Role: Android Developer
  • Tech Stack: Kotlin, Android
  • Year: 2025
Source Code

Project Overview

The app allows users to track their BMI (Body Mass Index) over time, with features for data entry, visualization, and analysis. The project was an opportunity to deepen my skills in Android development and UI/UX design.

  • User authentication and data storage
  • Interactive charts and data visualization
  • Modern Android UI with Kotlin
  • Focus on usability and accessibility

Challenges & Learnings

  • Learning Kotlin and Android best practices
  • Implementing data visualization in mobile apps
  • Ensuring responsive and accessible UI
  • Managing app state and user data securely
